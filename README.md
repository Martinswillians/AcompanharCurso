# ☕ Java COMPLETO — Tracker de Estudos

> Aplicação web para monitorar o progresso no curso **Java COMPLETO Programação Orientada a Objetos + Projetos** do instrutor **Nelio Alves** (Udemy).

🔗 **[Abrir o Tracker](https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO)**

---

## 📸 Visão Geral

O tracker foi desenvolvido para acompanhar todas as **263 aulas** organizadas em **20 seções** do curso, permitindo marcar o progresso, criar anotações por aula e registrar sessões de estudo.

---

## ✨ Funcionalidades

- ✅ **Marcar aulas como concluídas** — igual ao Udemy, aula por aula
- 📊 **Progresso visual** — barra por seção e percentual geral
- ✏️ **Editar títulos das aulas** — personalize conforme seu entendimento
- 📝 **Anotações por aula** — categorize em Dica, Dúvida, Importante ou Conceito
- 🔍 **Busca rápida** — encontre qualquer aula pelo nome ou número
- 📓 **Diário de estudos** — registre sessões com data, horas e descrição
- 💾 **Dados salvos localmente** — tudo persiste no `localStorage` do navegador
- 📱 **Responsivo** — funciona em desktop e celular
- 🌐 **100% offline** — sem dependências externas, sem login, sem servidor

## 🚀 Como usar

### Opção 1 — GitHub Pages (recomendado)

1. Faça um fork deste repositório
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `main` e pasta `/ (root)`
4. Acesse o link gerado pelo GitHub Pages

### Opção 2 — Uso local

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` (ou `java_tracker.html`) diretamente no navegador
3. Nenhuma instalação necessária
